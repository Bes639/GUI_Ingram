## ⚡ Особенности

- 🎨 **Modern Dark Theme** — стильный кастомный темный интерфейс в духе GitHub Dark, адаптированный под Tkinter.
- 🌐 **On-the-fly Локализация** — мгновенное переключение интерфейса между **Русским** и **Английским** языками в один клик.
- 🚀 **Потоковое асинхронное чтение** — ядро сканера работает в отдельном фоновом процессе; логи стримятся без задержек, фризов и зависаний интерфейса.
- 📟 **Умный Mini-Terminal** — встроенная консоль полностью поддерживает парсинг ANSI-цветов, отображает оригинальное ASCII-лого и корректно разделяет обычные логи (`\n`) от динамического статус-бара (`\r`).
- ⚙️ **Полный контроль CLI-флагов** — удобное управление всеми ключевыми параметрами сканирования через графические поля и чекбоксы.

## 🛠 Установка и запуск

### 1. Клонируй репозиторий со сканером и оболочкой
Убедись, что скрипт оболочки находится в корневой папке проекта Ingram (рядом с `run_ingram.py`).

```bash
git clone [https://github.com/Bes639/Ingram-GUI](https://github.com/Bes639/Ingram-GUI)
cd Ingram-GUI
2. ЗапускДля работы интерфейса используются стандартные библиотеки Python (tkinter), поэтому дополнительные тяжелые зависимости не требуются.Bashpython ingram_GUI.py
