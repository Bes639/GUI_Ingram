Установка и запуск
------------------

1. Клонируй репозиторий со сканером и убедись, что скрипт оболочки находится в корневой папке проекта Ingram (рядом с run_ingram.py).

git clone https://github.com/Bes639/Ingram-GUI
cd Ingram-GUI

2. Вытащи ingram_GUI.py и поставь его в папку, где находиться run_ingram.py, а папку Ingram-GUI можешь удалить

3. Зависимости для Linux (Ubuntu / Debian / Mint)
sudo apt install python3-tk
Для Linux (Fedora / Red Hat)
sudo dnf install python3-tkinter

4. Запуск
python ingram_GUI.py

Настройка параметров сканирования
---------------------------------
Интерфейс полностью заменяет ручной ввод флагов в терминале.

| Элемент GUI             | Флаг в CLI            | Описание |
|-------------------------|-----------------------|----------|
| Файл с целями           | -i / --in_file        | Путь к текстовому файлу со списком IP/диапазонов |
| Одна цель               | --target              | Сканирование одного хоста, CIDR-диапазона или IP:порт |
| Папка результатов       | -o / --out_dir        | Директория, куда Ingram сохранит отчеты и скриншоты |
| Порты                   | -p / --port           | Список целевых портов (можно разделять запятыми или пробелами) |
| Потоки                  | -t / --threads        | Количество параллельных потоков сканирования (по умолчанию: 150) |
| Таймаут                 | -T / --timeout        | Максимальное время ожидания ответа от хоста |
| Отладочный лог          | --debug               | Включает детализированное логирование для поиска проблем |
| Отключить скриншот      | -D / --disable_snapshot| Сканирование без создания графических снимков экранов |
